# concept

Living documents describing the emerging shape of the community or practice. These are **provisional** — expect rewrites.

## Docs

- `principles.md` — the philosophical/ethical grounding (to draft)
- `practices.md` — shared practices and rituals the group would use (to draft)
- `invitation.md` — who this is for and how one joins (to draft)

Start here when adding a new framing; move it to `research/` once it stabilises or becomes reference material rather than active thinking.
